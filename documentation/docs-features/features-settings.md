# Feature: Settings

The Settings module provides a persistent two-column interface for application configuration.

## Layout
- **Fixed Sidebar**: Permanent navigation rail on the left.
- **Modular View Area**: The right pane swaps between specialized sub-pages.

## Configurations
- **Status Display**: Customize Display Name and Character Class.
- **Account Profile**: Manage biography and perform a "Neural Wipe" to reset all local data.
- **Privacy & Notifications**: Manage haptic/audio feedback and encryption preferences (Boilerplate).
