## User Experience Flow
1. Unauthenticated user visits any protected route → redirected to `/auth/login`
2. User sees branded login screen with title and description
3. User clicks "Sign in with Google" → popup authentication
4. On success, loading spinner appears with "Redirecting to dashboard..." message
5. User automatically navigated to `/dashboard` with replace navigation (no back button loop)
