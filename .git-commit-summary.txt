LIVE TRANSCRIPTION FEATURE: Real-time voice capture feedback

### Summary
Implemented live transcription display during voice recording. Users now see
their speech transcribed in real-time every 3 seconds as they speak, replacing
the static "Listening..." text with dynamic, updating content.

### Technical Details

STREAMING STRATEGY:
- Audio chunks accumulate in transcriptionChunksRef every 3 seconds
- Each chunk converted to Blob → Base64 → Gemini API
- Response updates liveTranscription display
- Final complete audio still processed separately for structured output

DUAL BUFFER DESIGN:
- chunksRef: All audio → final complete audio (for structured output)
- transcriptionChunksRef: Periodic snapshots → live display (every 3s)
- Independent processing pipelines prevent interference

UI FEEDBACK:
- Transcription displays in italic text with left accent border
- "..." animation during API request (isTranscribing state)
- Cleared on recording stop and component unmount
- Never shown during isProcessing phase
